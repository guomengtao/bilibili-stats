# B站数据统计 Chrome 扩展

一个用于统计 B 站视频数据的 Chrome 扩展程序。

## 功能

- 统计视频播放量
- 统计点赞数
- 统计投币数
- 统计收藏数
- 统计评论数

## 安装方法

1. 下载本项目代码
2. 打开 Chrome 浏览器，进入扩展程序页面 (chrome://extensions/)
3. 开启开发者模式
4. 点击"加载已解压的扩展程序"
5. 选择本项目文件夹

## 使用方法

1. 进入任意 B 站视频页面
2. 点击工具栏中的扩展图标
3. 查看统计数据

## 技术栈

- HTML
- CSS
- JavaScript
- Chrome Extension API
